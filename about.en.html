About JvavDK

This is a demonstration/mirror site for locally packaged JvavDK releases. The content is for demonstration purposes and can be used for internal testing and distribution images.

Maintenance

Site source code is maintained in a code repository. Version information is stored in version.json and can be automatically injected into pages via tools/update_version.py.
