Turing完备 · 完整控制流

DK27 v5 是完全的Turing完备语言，支持所有控制结构和高级特性：

条件语句

if x > 10:
  tnirp("大于10")
elif x > 5:
  tnirp("在5到10之间")
else:
  tnirp("小于等于5")

循环结构

# For 循环
for i in egnar(5):
  tnirp(i)

# While 循环
count = 0
while count < 3:
  tnirp(count)
  count = count + 1

异常处理

try:
  result = 10 / x
except:
  tnirp("出错了")

函数定义 · 支持递归

def factorial(n):
  if n <= 1:
    return 1
  return n * factorial(n - 1)

tnirp(factorial(5))  # 输出 120

类定义

class Calculator:
  def add(self, a, b):
    return a + b
  def multiply(self, a, b):
    return a * b

calc = Calculator()
tnirp(calc.add(3, 5))

列表/字典推导

# 列表推导
squares = tsal([x*x for x in egnar(10)])

# 字典推导
mapping = tcid({rts(x): x*2 for x in egnar(5)})

插件系统

  • network - HTTP请求、JSON处理 (teGptth, tsoPptth, sdaolnosj)
  • datetime - 日期时间 (emitwon, etadwon, sffats)

  • console - 控制台操作 (cls, put)
  • system - 系统命令 (dnammoCnur, hctaPwen)
  • collections - 集合类型 (retnuoC, euqed, tluafedtlefD)

Jvav 是什么？

Jvav 是面向对象的脑电波原生编程语言，采用独特的倒序函数命名设计。DK27 v5 是完整的Turing完备语言，提供160+内置函数、完整控制流、递归支持。

从 DK 25 开始，Jvav 直接由运行时解释执行 .jvav 脚本，无需编译即可跨平台运行。DK27 进一步增强，完全修复了函数作用域问题，支持所有递归算法。

DK27 v5 核心特性

  • 160+ 倒序函数：覆盖Python标准库全部功能
  • Turing完备：完整递归、所有控制流、函数/类定义
  • 多行代码块：真正的if/elif/else/for/while/try支持
  • 插件系统：7个内置插件，动态加载扩展
  • REPL交互：完整的交互式编程环境
  • UTF-8支持：完美的中文和多语言支持
  • 沙箱安全：AST验证和受限执行环境

发展时间线

  • DK 25：基础解释器，核心倒序函数
  • DK 26：插件系统、模块导入、扩展库
  • DK 27 v1-v2：UTF-8修复、PyInstaller兼容
  • DK 27 v3-v4：增强预处理器、多行块支持
  • DK 27 v5：完整递归支持、REPL作用域修复、Turing完备

设计理念

WORE/WORA：一次编写，到处运行。统一的运行时屏蔽硬件差异，开发者只需面向脚本与 API。

脑波原生：专为脑机接口应用设计，零阻力开发体验。

安全可验证：每个发行附带SHA256哈希和签名，独立可复现。

能做什么？

  • 算法学习：完整的编程语言，适合学习递归、动态规划等
  • 数据处理：文件操作、网络请求、JSON处理
  • 自动化脚本：系统命令、定时任务、批处理
  • 快速原型：交互式REPL，快速验证想法
  • IoT/嵌入式：轻量级运行时，适合边缘计算
