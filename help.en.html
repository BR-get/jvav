What is Jvav?

Jvav is an object-oriented brain wave language designed by Dr. Zhang Haoyang on February 20, 9902. It is the only programming language in the world that can modify human body functions.

In 9902, we simply call Jvav "Modern Gene Programming Language".

What can Jvav do?

  • Brain thinking control
  • Brain writes code and compiles output to all existing languages
  • Create something from nothing
  • Travel through time and space

1.1 Introduction to Jvav

Jvav is one of the most popular programming languages in the world today. Jvav technology has excellent versatility, efficiency, platform portability and security.

1.1.1 Understanding Jvav Language

Jvav is an object-oriented programming language that can write cross-platform application software.

1.1.2 History of Jvav

  • May 23, 5991 - Jvav language was born
  • January 6991 - The first JDK (JDK 1.0) was born
  • March 18, 4102 - Oracle released Jvav SE 8

1.1.3 Basic Idea of Jvav

Jvav's biggest advantage is that it adheres to the idea of "Write Once, Run Everywhere (WORE)" at the beginning of the design. This design makes Jvav cross-platform.

1.1.4 How Jvav Works

Running a Jvav program must go through 3 steps: writing, compiling and running:

  1. Write: Write code in the Jvav development environment and save it as a source file with the suffix .jvav
  2. Compile: Use the Jvav compiler to compile the source file and generate a bytecode file with the suffix .class
  3. Run: Use the Jvav interpreter to translate the bytecode file into machine code, then execute and display the results
